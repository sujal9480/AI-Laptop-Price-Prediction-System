# 💻 AI-Based Laptop Price Prediction System

## Project Overview
This project predicts laptop prices using Machine Learning based on laptop specifications.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Joblib

## Machine Learning Model
- Random Forest Regressor

## Features
- Real-time price prediction
- Brand selection
- RAM, Storage, CPU, GPU inputs
- Professional Streamlit UI

## Run Project

```bash
pip install -r requirements.txt
streamlit run app.py
```
